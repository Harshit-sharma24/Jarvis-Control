# 🤖 Jarvis Control

Android + Python based remote PC control system.

## Features
- Mouse control
- Keyboard typing
- Live screen stream
- Screenshot capture
- Volume & brightness control
- Open apps remotely
- Touchpad mode

## Tech Used
- Python
- Kotlin
- XML
- PyAutoGUI
- MSS

## Setup

Install requirements:

pip install -r requirements.txt

Run server:

py -3.11 server.py

Connect Android app with PC IP.
